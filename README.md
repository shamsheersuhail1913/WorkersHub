# WorkerHub

WorkerHub is a modern platform connecting skilled workers with job opportunities. The platform consists of two main interfaces - one for workers and one for owners/employers.

## Features

- **Worker Portal**: Browse jobs, register profile, view employment opportunities
- **Owner Portal**: Post jobs, find skilled workers, manage hiring
- **Job Categories**: Agriculture, Plumbing, Electrical, Construction, Carpentry, Mechanics
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Real-time Notifications**: Get instant job alerts

## Files

- `worker-index.html` - Worker platform interface
- `owner-index.html` - Owner/Employer platform interface
- `index.html` - Main landing page

## Technologies Used

- HTML5
- React 18
- Tailwind CSS
- Babel (for JSX transformation)

## Getting Started

1. Clone the repository
2. Open any of the HTML files in your web browser
3. No installation or build process required!

## Features Breakdown

### For Workers
- Create your professional profile
- View available jobs in your area
- Filter jobs by skill type
- Set your expected daily wage
- Get notified about new opportunities

### For Owners
- Post job opportunities
- Search for skilled workers
- View worker profiles
- Manage hiring process

## Responsive Design

The platform is fully responsive and works on:
- Desktop computers
- Tablets
- Mobile phones
